# Expense Tracker 💸

A simple web-based expense tracker built with Flask.

## Features
- Add expenses with category and date
- Monthly filtering
- All-time total tracking
- Edit and delete expenses

## Tech Stack
- Python (Flask)
- SQLite
- HTML / CSS / JavaScript

## How to run

```bash
python app.py

Then open:
